#  StudySpot-Gaza
A smart Study Space Finder for Gaza University Students
# Project Overview |
StudySpot Gaza is a Flutter-based mobile application designed to help students in the Gaza Strip discover and navigate to available study and workspace locations. The app accounts for the unique infrastructure challenges of the region, specifically electricity and internet availability.
# Core Features | 
Study Hub Listings: Browse a curated list of study/work spaces.  
Real-Time Hub Info: View electricity availability, internet quality, and pricing.  
GPS & Map Integration: See nearby hubs on an interactive map and get directions.  
Search & Filter: Filter hubs by price, distance, or availability.
